## 🏢 About

**주식회사 별의문 (Byeoluimun / Stargate Corporation)**은 2025년 8월 15일 서울 강남구 대치동에서 설립된 AI 소프트웨어 기업입니다. 4대 사업축(**AI Tutoring · Digital Products · Short-term Rental · E-commerce**)을 중심으로 교육·자동화·데이터 기반 사업을 전개합니다.
